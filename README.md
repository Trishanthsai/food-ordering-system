# Food Ordering System

A full stack Food Ordering System project with frontend and backend.

## Frontend

* JavaScript
* HTML
* CSS
* Axios

## Backend

* Spring Boot
* Spring Security
* MySQL
* JWT Authentication
* JPA / Hibernate

## Features

* User Registration and Login
* JWT-based Authentication
* Separate Admin and User Roles
* Add, Update and Delete Food Items
* Browse Food Menu
* Add Items to Cart
* Place Orders
* Payment Handling
* Order Tracking
* Secure Backend APIs

## Run Frontend

```bash id="pdd1a6"
npm install
npm start
```

Frontend runs on:


http://localhost:3000


## Run Backend

mvn clean install
mvn spring-boot:run


Backend runs on:


http://localhost:8080


## Database
* Postgresql
